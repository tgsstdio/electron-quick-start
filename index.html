mg-asset-wrangler

We are using Node.js , Chromium , and Electron .

Steps

  1. Pick a block structure
  2. Pick an operating system
  3. Select a destination folder
  4. View existing assets
  5. Select a source file
  6. Generate an asset identifier
  7. Now click Insert file
  8. Done
